# Calc with *GO*

Here I use variable to save numbers and do the aritmethic.

- **How to use calc?**

make sure you run `main.go` in your terminal

```bash
go run main.go
```
<br>

- **Example of var `a` and `b`**
```go
a := 2
b := 2
```
<br>

- **Expected Output**
```go
Add: 4
Sub: 0
Mul: 4
Div: 1
```
